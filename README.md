# README - CS4337 Project 1

## Project Description
This project is a prefix-notation expression calculator implemented in Racket. It supports binary operations (+, *, /), unary negation (-), and a history system where previous results can be referenced ($id) in the expression.

## Files and Roles
* project1.rkt: The main source code file.
* devlog.md: The development log documenting each session's progress.

## How to Run the Program
The code can be in either Interactive or Batch mode.

# Interactive Mode
To run the calculator with user prompts:
"racket project1.rkt"

# Batch Mode
To run the program without prompts, use the -b or --batch flag:
"racket project1.rkt -b"

## Quitting
To exit the program, type "quit" at the prompt.
